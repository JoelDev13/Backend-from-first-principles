# Roadmap de Backend desde Primeros Principios

Bienvenido a este repositorio, que documenta mi **aprendizaje de backend desde cero**, siguiendo la playlist de [Sriniously](https://www.youtube.com/playlist?list=PLui3EUkuMTPgZcV0QhQrOcwMPcBCcd_Q1) sobre ingeniería backend.  
El objetivo es **comprender cómo funciona el backend realmente**, aplicar los conceptos en mini-proyectos
---

## Contenido del repositorio

1. **Notas por video** (`/notes`)  
   - Archivos `.md` con resumen de cada video, conceptos clave, ejemplos de código y reflexiones  
   - Cada archivo corresponde a un video de la playlist y resume los conceptos más importantes

2. **Proyectos de práctica** (`/projects`)  
   - Mini-proyectos donde aplico los conceptos aprendidos en la playlist

---

## Propósito del repositorio

- **Aprender desde primeros principios:** entender cómo funciona realmente cada parte del backend, en lugar de solo copiar ejemplos
- **Documentar el aprendizaje:** notas claras y organizadas que pueden servir como referencia futura
